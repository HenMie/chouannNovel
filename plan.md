# 前端优化与开发计划 (Optimization Plan)

基于最新代码库梳理，整理已完成事项并明确后续优先级。

---

## 已完成（简述）
- [x] 性能：列表虚拟化组件、图片懒加载/资源压缩、请求防抖节流、构建去 console、数据库高频字段索引。
- [x] 核心：工作流导入/导出、项目备份/恢复、工作流版本历史。
- [x] 编辑器：撤销/重做、跨工作流复制粘贴、多选批量操作、右键菜单、快捷键支持。
- [x] 设定库：虚拟列表+搜索筛选、批量启用/禁用/删除、注入模板编辑、设定库导入/导出（合并/替换）。
- [x] 执行：执行历史列表、节点流式输出、暂停/继续/取消、执行记录导出 TXT/MD。
- [x] 引导与帮助：新手引导 Tour、快捷键面板（F1）。
- [x] 错误与日志：AppErrorBoundary、全局错误监听、统一错误日志 logError。
- [x] 类型与测试：TypeScript 严格模式、移除 @ts-ignore；lib/hooks/UI 单测 + Playwright E2E（coverage 已生成）。

## 待办与规划

### 设定库增强
- [ ] 设定模板：常用角色/世界观等预设，一键套用。
- [ ] 设定关联：条目间的引用关系展示与跳转。

### 执行增强
- [ ] 执行队列：支持排队/并行调度与队列状态管理。
- [ ] 定时执行：计划任务（如 cron/一次性）与重试策略。
- [ ] 断点调试：节点级暂停、变量快照查看与手动修改。

### 可视化与体验
- [ ] 工作流可视化视图：流程/数据流图模式切换。
- [ ] 数据流指示与进度：执行进度条、节点间数据路径高亮。
- [ ] 仪表盘图表：执行次数、字数等趋势统计。
- [ ] 拖拽上传：项目或工作流文件的拖拽导入。
- [ ] 多标签页与布局记忆：同时打开多个工作流、记住面板尺寸/折叠状态。

### 性能与质量
- [ ] 性能监控埋点：关键操作耗时、首屏/交互指标采集。
- [ ] 测试补强：扩大 engine/stores 边界用例、导入导出异常场景；补充组件回归测试。
